Detailed Description

Represents a single card in the detective sheet (e.g. Ms Scarlett) The sheet UI has fixed number of player columns 6 rest columns will be inactive.

For each column this entry tracks three things: active: is there player in this col autoCrossedOut: the game auto marked this card manualCrossedOut: the human player ticked the cross button
