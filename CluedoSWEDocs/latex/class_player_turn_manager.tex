\doxysubsection{Detailed Description}
Attach to:\+ Character Panel. 

No Game\+Manager needed. Completely self-\/contained.

Highlights the active player\textquotesingle{}s portrait with a bright yellow outline. Uses Unity\textquotesingle{}s built-\/in Outline component — added automatically to each character Game\+Object so you don\textquotesingle{}t need to add anything manually.

SETUP:\+
\begin{DoxyEnumerate}
\item Select "{}\+Character Panel"{} in the Hierarchy.
\item Add Component → \doxylink{class_player_turn_manager}{Player\+Turn\+Manager}.
\item Set the Size of the Characters array to however many players are in your game (up to 6), then drag each character Game\+Object in:\+ \+[0]\+ Miss\+Scarlett \+[1]\+ Reverend\+Green \+[2]\+ Colonel\+Mustard \+[3]\+ Mrs\+White \+[4]\+ Mrs\+Peacock \+[5]\+ Professor\+Plum
\item Drag "{}\+End Turn Button"{} (inside Options Panel) into the End Turn Button slot. The button stays where it is — this is just a reference so this script can listen for clicks.
\item Set \doxylink{class_player}{Player} Count to how many players are in this game (1-\/6). Only that many portraits will be shown; the rest are hidden.
\end{DoxyEnumerate}
